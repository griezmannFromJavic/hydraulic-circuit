implement Depth-First Search (DFS) to create a tree

PSEUDOCODE:
function DFS(graph, current_vertex, tree):
    mark current_vertex as visited
    add current_vertex to tree
    
    for each neighbor in graph.adjacency_list[current_vertex]:
        if neighbor is not visited:
            create an edge (current_vertex, neighbor) in the tree
            recursively call DFS(graph, neighbor, tree)
